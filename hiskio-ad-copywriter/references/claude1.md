​  
🔵 藍色｜最常使用  
/init → 建立 CLAUDE.md  
/context → 查看 context 使用量  
esc → 取消當前操作  
esc \+ esc → 回溯 prompt 歷史  
@file → 在 prompt 中引用檔案  
\[image\] → 複製貼上圖片  
/compact → 壓縮 context  
/clear → 清除 context  
​  
🟣 紫色｜必要時使用  
shift \+ tab → Plan Mode  
Ultrathink → 處理複雜任務  
Ctrl \+ T → 顯示 ToDo List  
​  
🟢 綠色｜設定時使用  
/login /logout /model → 帳號與模型設定   
/ide → 編輯器整合  
/permissions → 管理工具權限  
/custom → 自訂指令
